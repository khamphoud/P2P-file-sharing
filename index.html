P2P File Transfer

Send files directly between devices using a 4-digit code

Enter a 4-digit code and select files to begin
Share this code with the sender

How to use:

  1. On the receiving device, go to "Receive Files" and share the 4-digit code
  2. On the sending device, go to "Send Files", enter the code and select files
  3. Click "Connect & Send" to establish connection and transfer files
  4. Files are transferred directly between devices - no servers involved
